# 모행 Frontend (모두의 모든 행사)

Cloud Native 기반 통합 이벤트 관리 플랫폼  
사용자 인터페이스 및 사용자 경험을 담당하는 React 기반 프론트엔드

---

## 프로젝트 소개

모행은 누구나 행사를 주최하고 참여할 수 있는 플랫폼입니다.  
본 저장소는 사용자 UI 및 서비스 흐름을 담당하는 프론트엔드입니다.
